# Взлом школьного дневника 

Данное приложение содержит в себе функции для изменения содержания электронного дневника.

## Как работает

Есть 3 функции.

### 1. Получение имени ученика

Получает полную информацию об ученике по его имени и/или фамилии.

### 2. Исправление оценок

Находит все оценки ученика, которые ниже 4 (`points__in=[2, 3]`), и заменяет их на 5.

### 3. Удаление замечания 

Удаляет все записи вида "Замечание" из дневника ученика.

### 4. Добавление похвалы

Создаёт новую запись `Commendation` с случайной похвалой, датой урока и именем учителя.

## Использование

Для использования потребуеться положить файл БД дневника в директорию со скриптом. Далее необходимо запустить командную строку и перейти в директорию со скриптом. Затем пишем команду: 

```
from scripts import fix_marks, remove_chastisements, create_commendation
```

Для использованя введите команду и подставте нужные значения. Образец команд и пример значений: 

+ Команда `get_schoolkid("Иван Фролов)` получает полную информацию об Иване Фролове (не выводит ничего в консоль).
+ Команда `fix_marks("Иван Фролов)` исправит все плохие оценки у Ивана Фролова.
+ Команда `remove_chastisements("Иван Фролов")` удалит все замечания, в адрес Ивана Фролова.
+ Команда `create_commendation("Иван Фролов", "Музыка")` добавит случайную похвалу Ивану Фролову за урок музыки.
